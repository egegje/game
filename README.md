# Game — Тетрис + Банкир (game.eg.je)

Мобильное веб-приложение с двумя играми по дизайну «Плитка»: Тетрис и Банкир
(симулятор банка для детей 8+). Светлая и тёмная темы, прогресс в localStorage.

- `index.html` + `css/` + `js/` — SPA, хостится на GitHub Pages (game.eg.je)
- `ios/` — нативная iOS-обёртка (SwiftUI + WKWebView), сборка: `cd ios && xcodegen generate && open GameApp.xcodeproj`
